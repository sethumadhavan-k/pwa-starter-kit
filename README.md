## PWA Starter Kit
PWA Starter Kit is basic stucture for starting a PWA App

#### Folder Stucture


```
├── assets 
│   └── css
│        └─ style.css
│   └── js
│        └─ app.css
│   └── img
│        └─ icons
├── index,html 
├── manifest.json
├── sw.js 
```
